Demostración: box-sizing (content-box vs border-box)

#example1
content-box

width:300px + padding 30px + border 10px ➔ ancho TOTAL = 300+60+20 = 380px (se desborda)

#example2
border-box

El padding y border se incluyen en el width: 300px → se mantiene 300px total.

El selector universal * aplica box-sizing: border-box a TODO el documento para evitar sorpresas. Pero #example1 lo sobreescribe con content-box para mostrar la diferencia. Observa cómo el primer bloque se ensancha más allá de 300px.
